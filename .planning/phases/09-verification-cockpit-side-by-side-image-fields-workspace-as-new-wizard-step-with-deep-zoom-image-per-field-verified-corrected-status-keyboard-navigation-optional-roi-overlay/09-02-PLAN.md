    Renders a full-height horizontal flex container split into two panes with a drag handle.
    - Props: `left: React.ReactNode`, `right: React.ReactNode`
    - Reads initial split percent from `useWizardStore(s => s.cockpitSplitPercent)` (default 50)
    - The drag handle is a 4px-wide vertical divider with a `cursor-col-resize` class
    - On mousedown on the divider: register mousemove + mouseup on `document`; set `document.body.style.userSelect = 'none'` to prevent text selection during drag
    - During mousemove: compute new split percent from `(e.clientX / containerRef.clientWidth) * 100`, clamp to [20, 80] (minimum pane widths)
    - On mouseup: call `setCockpitSplitPercent(newPercent)` to persist via Zustand; restore `document.body.style.userSelect = ''`
    - Left pane: `style={{ width: \`${splitPercent}%\` }}` with `overflow: hidden`
    - Right pane: `style={{ flex: 1 }}` with `overflow-y: auto`
    - The component uses a local `splitPercent` state initialized from the Zustand value; writes back to Zustand only on mouseup (not during drag to avoid store thrash)
    - Apply Parchment theme: the drag handle can be `bg-archive-300 hover:bg-archive-500` or similar subtle border color from the project's tailwind palette

    CREATE `apps/frontend/src/features/verify/ImagePane.tsx`:
    Renders the full-res JPEG for the current card with wheel-zoom + drag-pan + double-click-reset.
    - Props: `imageUrl: string` (constructed as `/batches-static/${batchName}/${filename}` — confirmed URL pattern from ThumbnailCell.tsx line 12)
    - Container div with `overflow: hidden`, `w-full`, `h-full`, `relative`, `bg-stone-100`
    - Inner `<img ref={imgRef}` with `style={{ transformOrigin: 'center center', transition: 'none', maxWidth: 'none', maxHeight: 'none', display: 'block' }}`
    - State managed via `useRef` (NOT `useState`) for scale/tx/ty to avoid re-renders on every frame:
      ```typescript
      const scale = useRef(1);
      const tx = useRef(0);
      const ty = useRef(0);
      const imgRef = useRef<HTMLImageElement>(null);
      const containerRef = useRef<HTMLDivElement>(null);

      const applyTransform = () => {
        if (imgRef.current) {
          imgRef.current.style.transform =
            `scale(${scale.current}) translate(${tx.current}px, ${ty.current}px)`;
        }
      };
      ```
    - WHEEL ZOOM — CRITICAL: use `addEventListener` with `{ passive: false }`, NOT the JSX `onWheel` prop (React's onWheel is passive by default in React 17+ and `e.preventDefault()` is ignored):
      ```typescript
      useEffect(() => {
        const el = containerRef.current;
        if (!el) return;
        const handleWheel = (e: WheelEvent) => {
          e.preventDefault();
          const factor = e.deltaY > 0 ? 0.9 : 1.1;
          scale.current = Math.max(0.5, Math.min(8, scale.current * factor));
          applyTransform();
        };
        el.addEventListener('wheel', handleWheel, { passive: false });
        return () => el.removeEventListener('wheel', handleWheel);
      }, []);
      ```
    - DRAG PAN — track mousedown/mousemove/mouseup on the container:
      ```typescript
      const dragging = useRef(false);
      const lastMouse = useRef({ x: 0, y: 0 });

      const handleMouseDown = (e: React.MouseEvent) => {
        dragging.current = true;
        lastMouse.current = { x: e.clientX, y: e.clientY };
        document.body.style.userSelect = 'none';
      };
      const handleMouseMove = (e: React.MouseEvent) => {
        if (!dragging.current) return;
        tx.current += (e.clientX - lastMouse.current.x) / scale.current;
        ty.current += (e.clientY - lastMouse.current.y) / scale.current;
        lastMouse.current = { x: e.clientX, y: e.clientY };
        applyTransform();
      };
      const handleMouseUp = () => {
        dragging.current = false;
        document.body.style.userSelect = '';
      };
      ```
    - DOUBLE-CLICK RESET:
      ```typescript
      const handleDoubleClick = () => {
        scale.current = 1; tx.current = 0; ty.current = 0;
        applyTransform();
      };
      ```
    - Attach onMouseDown, onMouseMove, onMouseUp, onDoubleClick to the container div via JSX props (these are fine as JSX — only wheel needs addEventListener).
    - Show a small overlay hint "Scroll to zoom · Drag to pan · Double-click to reset" in `text-xs text-archive-ink/50 absolute bottom-2 right-2` that disappears after first interaction (optional — Claude's discretion on UX detail).
    - When `imageUrl` prop changes (new card selected), reset transform: scale=1, tx=0, ty=0.
